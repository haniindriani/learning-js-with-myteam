Pengenalan JavaScript

Halaman ini dimuat untuk penjelasan dari pengenalan JavaScript

JavaScript adalah bahasa pemograman dan merupakan bahasa pemograman terpopuler

Lalu kenapa belajar JavaScript

JavaScript adalah salah satu dari 3 bahasa yang dimana semua web developer harus belajar diantaranya :

  1. HTML untuk menentukan isi konten dari sebuah halaman web
  2. CSS untuk menentukan tata letak dari sebuah halaman web
  3. JavaScript untuk memprogram prilaku dari sebuah halaman web

Untuk analoginya

Web itu ibarat manusia yang memiliki tulang, kulit, dan otak. Tulang itu ibarat HTMLnya, kulit ibarat CSS, dan JavaScript itu ibarat otak manusia yang dimana otak manusia bisa berprilaku untuk berjalan, berlari, dan lain sebagainya
